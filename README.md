# Water-Tank-Ladder
CCW project connected with Factory IO to run a water filling tank based off a 10 second timer. 

Fill button = fills for 10 seconds 
Discharge button = discharges for 10 seconds 

sensors at top and bottom to prevent overflow and discharging with no water in tank. 
Emergency stop button implemented. 
Lights for fill, discharge, and idle state.

Seamless transition from fill to discharge state.
